## ‚úÖ IDP Checklist

### At IDP Creation
- [ ] Completed collaboratively with IC
- [ ] Goals are SMART (Specific, Measurable, Achievable, Relevant, Time-bound)
- [ ] Goals align with career framework
- [ ] Resources and support identified
- [ ] Timeline and milestones clear
- [ ] Success criteria defined
- [ ] IC has copy of document

### Monthly
- [ ] Review progress in 1:1
- [ ] Document wins and challenges
- [ ] Collect evidence and feedback
- [ ] Adjust goals if needed
- [ ] Provide coaching and support

### Quarterly
- [ ] Complete thorough review
- [ ] Celebrate achievements
- [ ] Update skills matrix
- [ ] Create next quarter's goals
- [ ] Refresh IDP document

### Annually
- [ ] Comprehensive year-in-review
- [ ] Update long-term career vision
- [ ] Assess promotion readiness if relevant
- [ ] Set annual priorities
