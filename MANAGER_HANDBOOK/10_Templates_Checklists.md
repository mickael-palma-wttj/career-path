## Essential Checklists

### Checklist: Before Promotion

**3-6 Months Before:**
- [ ] Review framework criteria for next level
- [ ] Identify specific gaps in IC's current performance
- [ ] Create development plan targeting those gaps
- [ ] Assign stretch projects demonstrating next-level work
- [ ] Set clear timeline for promotion consideration

**1-3 Months Before:**
- [ ] IC consistently demonstrating next-level work for 6+ months
- [ ] Collect peer feedback from 5-8 collaborators
- [ ] Document specific examples of next-level impact
- [ ] Review with manager/skip-level for calibration
- [ ] Confirm budget and approval process

**Promotion Decision Time:**
- [ ] Complete promotion packet with evidence
- [ ] Map achievements to framework criteria
- [ ] Include peer feedback summary
- [ ] Quantify business impact
- [ ] Submit through proper channels
- [ ] Prepare for calibration session
- [ ] Have backup plan if delayed

**After Promotion:**
- [ ] Celebration and recognition (public announcement)
- [ ] Updated responsibilities and expectations
- [ ] New development plan for current level
- [ ] Adjust compensation/title
- [ ] Check in monthly for first quarter

**See:** [Promotion Readiness guide](06_Promotion_Readiness.md)

---

### Checklist: End of Year Review

**4-6 Weeks Before:**
- [ ] Review IC's goals and development plan from start of year
- [ ] Collect peer feedback (5-8 people)
- [ ] Review all 1:1 notes and documented feedback
- [ ] Assess performance against framework criteria
- [ ] Draft preliminary ratings and justification
- [ ] Identify key achievements and growth areas

**2-3 Weeks Before:**
- [ ] Request IC's self-assessment
- [ ] Compare your assessment to theirs
- [ ] Prepare specific examples for each rating
- [ ] Draft written review
- [ ] Calibrate with other managers
- [ ] Review compensation changes (if applicable)
- [ ] Prepare promotion discussion (if relevant)

**Review Meeting:**
- [ ] Schedule 60-90 min meeting
- [ ] Review overall performance and ratings
- [ ] Discuss specific strengths and achievements
- [ ] Address development areas with examples
- [ ] Set goals for next year
- [ ] Discuss career progression
- [ ] Review compensation changes
- [ ] Document IC's reactions and questions

**After Meeting:**
- [ ] Submit final review through HR system
- [ ] Update development plan for new year
- [ ] Schedule follow-up on any open items
- [ ] Document agreed-upon next steps

**See:** [Feedback Delivery guide](05_Feedback_Delivery.md)

---

### Checklist: New Hire Onboarding

**Before Day 1:**
- [ ] Equipment and accounts set up
- [ ] Onboarding buddy assigned
- [ ] First week schedule planned
- [ ] Welcome email sent
- [ ] Team announcement prepared

**Week 1:**
- [ ] Welcome meeting (30-60 min)
- [ ] Introduction to team
- [ ] Overview of career framework
- [ ] Set initial learning goals
- [ ] Schedule regular 1:1s (weekly for first month)
- [ ] Assign first small project

**Month 1:**
- [ ] Review career framework in detail
- [ ] Discuss their background and goals
- [ ] Create initial development plan
- [ ] Set 3-month goals
- [ ] Regular feedback on early work
- [ ] Check in on team integration

**Month 3:**
- [ ] 90-day review conversation
- [ ] Assess progress against goals
- [ ] Collect peer feedback
- [ ] Refine development plan
- [ ] Adjust to bi-weekly 1:1s (if appropriate)

**Month 6:**
- [ ] Formal 6-month review
- [ ] Assess level calibration
- [ ] Set annual goals
- [ ] Discuss long-term career path

**See:** [Getting Started guide](00_Getting_Started.md)

### Checklist: Retention Conversation

**When at-risk IC signals unhappiness:**

**Immediate (Day 1-2):**
- [ ] Schedule extended 1:1 (60-90 min) ASAP
- [ ] Create safe space for honest conversation
- [ ] Express genuine concern and desire to help
- [ ] Listen without getting defensive

**Assessment (Day 2-5):**
- [ ] Understand specific issues and root causes
- [ ] Determine if issues are fixable
- [ ] Assess how critical they are to retain
- [ ] Evaluate what you can actually offer
- [ ] Get manager/leadership input
- [ ] Review budget for compensation adjustments

**Response Building (Day 5-10):**
- [ ] Build comprehensive retention package
- [ ] Get necessary approvals (comp, promotion, etc.)
- [ ] Prepare specific timeline commitments
- [ ] Identify quick wins vs. longer-term changes
- [ ] Be realistic about what you can't do

**Response Conversation:**
- [ ] Present clear, specific offer
- [ ] Explain what you can and can't do
- [ ] Give them time to consider (few days)
- [ ] Ask for commitment if they stay
- [ ] Respect their decision either way

**If They Stay:**
- [ ] Deliver on promises immediately
- [ ] Show visible progress within first week
- [ ] Check in weekly for first month
- [ ] Monthly check-ins for next 3 months
- [ ] Monitor engagement closely
- [ ] Sustain changes long-term

**If They Leave:**
- [ ] Support graceful transition
- [ ] Ensure proper knowledge transfer
- [ ] Maintain positive relationship
- [ ] Learn from experience
- [ ] Don't take it personally

**See:** [Retention Strategies guide](08_Retention_Strategies.md) and [The Retention Crisis case study](../CASE_STUDIES/The_Retention_Crisis.md)

### Checklist: Difficult Feedback Delivery

**Preparation:**
- [ ] Identify specific behavior or issue
- [ ] Collect concrete examples (dates, situations)
- [ ] Understand impact on team/work/business
- [ ] Plan using SBI framework (Situation-Behavior-Impact)
- [ ] Consider IC's perspective
- [ ] Prepare for defensive reactions
- [ ] Schedule private 1:1 (30-45 min)

**During Conversation:**
- [ ] Start with intent: "I want to help you succeed"
- [ ] Be direct and specific
- [ ] Use SBI framework
- [ ] Give examples
- [ ] Listen to their response
- [ ] Acknowledge their perspective
- [ ] Focus on forward action
- [ ] Document the conversation

**After Conversation:**
- [ ] Send follow-up email summarizing discussion
- [ ] Set clear expectations and timeline
- [ ] Schedule follow-up check-in
- [ ] Provide support/resources
- [ ] Monitor for improvement
- [ ] Give positive feedback when you see progress

**Follow-Up:**
- [ ] Check in at agreed timeline
- [ ] Acknowledge progress or escalate if no improvement
- [ ] Continue feedback loop
- [ ] Document ongoing conversations

**See:** [Feedback Delivery guide](05_Feedback_Delivery.md)

### Checklist: Performance Improvement Plan (PIP)

**Before Starting PIP:**
- [ ] Documented history of performance issues
- [ ] Previous feedback and coaching attempts
- [ ] Partner with HR on PIP process
- [ ] Clear business justification
- [ ] Manager/leadership alignment
- [ ] Legal review (if needed)

**PIP Document:**
- [ ] Specific performance problems documented
- [ ] Clear expectations for improvement
- [ ] Measurable success criteria
- [ ] Timeline (typically 60-90 days)
- [ ] Support and resources provided
- [ ] Consequences if unsuccessful
- [ ] Regular check-in schedule (weekly)

**During PIP:**
- [ ] Weekly progress check-ins
- [ ] Document all conversations
- [ ] Provide specific feedback
- [ ] Offer support and resources
- [ ] Track improvements or lack thereof
- [ ] Partner with HR throughout

**PIP Conclusion:**
- [ ] Final assessment against success criteria
- [ ] Decision: successful completion, extension, or termination
- [ ] Documentation of outcome
- [ ] HR partnership on next steps
- [ ] If successful: clear ongoing expectations
- [ ] If unsuccessful: termination process with HR

**See:** [PIP Template](../TOOLS_TEMPLATES/PIP_Template.md) and [How to Address Underperformance](../PRACTICAL_GUIDES/How_to_Address_Underperformance.md)

### Checklist: Team Transition (New Manager)

**First Week:**
- [ ] 1:1 with each team member (60 min each)
- [ ] Understand their role, goals, and concerns
- [ ] Review existing development plans
- [ ] Ask: "What do you need from me as your manager?"
- [ ] Observe team dynamics
- [ ] Review current projects and priorities

**First Month:**
- [ ] Establish regular 1:1 cadence
- [ ] Understand the career framework
- [ ] Review each IC's performance history
- [ ] Set your management philosophy and expectations
- [ ] Identify quick wins and issues
- [ ] Begin building relationships

**First 3 Months:**
- [ ] Complete assessment of each IC
- [ ] Update all development plans
- [ ] Address any urgent issues
- [ ] Establish team norms and culture
- [ ] Set quarterly goals
- [ ] Build trust and credibility

**See:** [Getting Started guide](00_Getting_Started.md)

## How to Use These Templates

### 1. **Start with the Template**
Copy the relevant template and customize for your situation.

### 2. **Adapt to Context**
Every person and situation is unique. Use templates as starting points, not rigid scripts.

### 3. **Document Consistently**
Using templates ensures you capture important information and maintain consistency across team.

### 4. **Review and Improve**
After using a template, note what worked and what didn't. Improve over time.

### 5. **Share with Team**
Consider sharing relevant templates with ICs (like development plan template) to set clear expectations.
